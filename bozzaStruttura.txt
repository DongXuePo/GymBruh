config = config.php è uno dei file più importanti in un progetto PHP: contiene tutte le impostazioni globali del sito,
         cioè le variabili e le configurazioni comuni che altri file devono conoscere. 

avvio sessione
include del database
costanti
funzioni condivise
impostazioni del server

in ognia pagina: " require "config.php" "
        
<?php
// AVVIO SESSIONE
if (session_status() === PHP_SESSION_NONE) {
    session_start();
}

// NOME DEL SITO E URL BASE
$site_name = "GymBruh";
$base_url = "http://localhost/progettoGym/";

// MOSTRARE ERRORI (solo in sviluppo)
error_reporting(E_ALL);
ini_set('display_errors', 1);

// IMPOSTAZIONI GENERALI
date_default_timezone_set("Europe/Rome");

// INCLUDO FILE NECESSARI
require_once __DIR__ . "/db.php";
require_once __DIR__ . "/functions.php";
?>
